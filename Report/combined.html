Document merging — key challenge for this team

  • When multiple sources are uploaded together without a correctly configured template, content can appear across sections — the team identified this as a template setup issue and are working with Beam to resolve it
  • Some clinicians who dictate directly found the current template layout required more reorganisation than their existing workflow — this is being addressed through template iteration
  • ADOS sessions: the tool sometimes generalised communication ability from limited cues, losing clinically significant nuance
  • A blank template / plain document output option was the most commonly requested improvement

05

What both pilots agree on

Clinicians are more present in sessions when they are not simultaneously taking notes — and families notice.
Write-up time is significantly reduced for most task types, particularly new patient assessments and standalone reports.
Parents and young people feel more heard when the clinician is engaged rather than typing.
The tool works best for live session capture and transcription — upload-and-merge workflows are more complex and variable.
Accuracy requires checking — clinicians must review and edit all AI output before it is finalised.
Templates need to match local practice. Generic outputs take longer to correct than to write from scratch.
06

Challenges and limitations

Panel template setup (HD Autism) — when multiple documents are uploaded together, content can spread across sections if the template isn't configured to keep sources separate. The team have identified this as a template setup issue rather than a system limitation, and are working with Beam to refine the workflow.
Welsh language (CTM) — hallucinations observed in Welsh-medium sessions. The tool is not currently safe for Welsh-only clinical use.
Medication names (HD ADHD) — hallucinated medication names and dosages flagged in ADHD review sessions. All medication references must be independently verified.
ADOS nuance (HD Autism) — the tool sometimes generalises communication ability from limited cues, losing clinically significant detail needed for diagnosis.
Technical access — login passcode delays, phone screen locking mid-recording, and intermittent connectivity affected sessions across both sites.
Dictation-first workflows (HD Autism) — clinicians who dictate directly found some tasks took longer with Magic Notes, as the structured output required extensive reformatting.
07

Recommendations

For Magic Notes / Beam

● Blank / plain document output — requested by multiple HD clinicians; would significantly improve usability for upload workflows
● Panel template refinement — work with the Hywel Dda Autism team to configure templates so multi-source uploads populate the correct sections reliably
● Welsh-language accuracy — essential before NHS Wales-wide rollout; hallucinations in Welsh-medium sessions are a clinical safety concern
● Flag low-confidence outputs — clinicians need to know when the tool is uncertain, not just when it is confident; especially important for medication and diagnostic detail

For NHS Wales / service leads

✓ Confirm IG status under WHC guidance 2025/026 before wider rollout
✓ Do not mandate use for multi-source panel preparation until document merging is resolved
✓ Plan for a 3–5 session learning curve; allocate protected time for new users to build confidence
✓ Share template libraries across health boards — avoid duplication of development effort
